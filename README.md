# To-Do List (Python)

This is a simple console-based To-Do List program built using Python.

## Features
- Add new tasks
- View all tasks
- Delete a task using task number
- Mark tasks as completed
- Continuous menu-based interaction
- Basic input validation

## Concepts Used
- Python Lists
- Loops (while, for)
- Conditional statements
- List indexing
- List methods (append, pop)
- User input handling
